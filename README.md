
# Smart Darts Score Keeper

Smart Darts Score Keeper is a browser-based web application for tracking scores in darts games (301 up and 501 up). Built with TypeScript, it features a modern, responsive, and accessible UI for real-time scorekeeping and game management for two players.

## Features
- Add two players and set their names
- Select game type: 301 up or 501 up
- Choose set size (number of legs to win)
- Enter three hits per turn
- View turn-by-turn history for each player
- Edit or remove the last turn
- Real-time display of each player’s score, legs won, average, and highest turn
- Reset leg scores after each leg and declare the winner
- Light and dark theme toggle
- Responsive and accessible design

## Getting Started
1. Clone the repository:
	```
	git clone <repo-url>
	cd smart-darts-score-keeper-group-12
	```
2. Install dependencies:
	```
	npm install
	```
3. Build the project:
	```
	npm run build
	```
4. Start the development server:
	```
	npm start
	```
5. Open your browser and go to [http://localhost:3000](http://localhost:3000)

## Usage
1. Enter player names, select game type and set size, then start the game.
2. Enter three hits per turn for the active player.
3. Use Edit/Remove to correct mistakes in the last turn.
4. Pause or exit the game at any time.
5. Toggle between light and dark themes.
